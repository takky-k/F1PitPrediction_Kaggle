# Data

This document is written in both English and Japanese. English appears first, followed by Japanese translation.

## English

Place the Kaggle competition files here after downloading them locally:

```text
train.csv
test.csv
sample_submission.csv
```

This workspace also used an optional external/original file:

```text
f1_strategy_dataset_v4.csv
```

The CSV files are intentionally ignored by git. This keeps the repository light and avoids redistributing competition data.

## 日本語

この文書は英語と日本語の両方で書いています。最初に英語、そのあとに日本語訳を置いています。

Kaggle からダウンロードしたコンペデータを、このフォルダに置きます。

```text
train.csv
test.csv
sample_submission.csv
```

この作業フォルダでは、任意の外部/元データとして次のファイルも使いました。

```text
f1_strategy_dataset_v4.csv
```

CSV ファイルは意図的に git の対象外にしています。リポジトリを軽く保ち、コンペデータを再配布しないためです。
